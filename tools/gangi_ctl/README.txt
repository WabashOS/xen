DEPENDENCIES

Besides Xen libraries, this tool requires:

- g++

So far it compiles with g++ 4.8.2 on Ubuntu 14.04. 

- Multi-word Bitmask Library (libbitmask1)

To install in Ubuntu 14.04, use the command:

$ sudo apt-get install libbitmask1 libbitmask-dev
